# Cambrón Studio — LinkedIn Strategy Dashboard

Dashboard navegable con toda la estrategia de LinkedIn para el perfil de **Laura Cambrón Sánchez**, CEO & Fundadora de Cambrón Studio.

Trabajo grupal — Corporate Personal Branding en LinkedIn.

---

## Estructura del repositorio

```
/
├── index.html     # Estructura HTML completa (sin estilos ni JS inline)
├── styles.css     # Todos los estilos — design tokens, layout, componentes
├── app.js         # Lógica JS — datos de posts, navegación, búsqueda, copiar
└── README.md      # Este archivo
```

---

## Cómo usar

1. Clona el repositorio o descarga los tres archivos en la misma carpeta
2. Abre `index.html` en cualquier navegador moderno
3. No requiere servidor, dependencias ni build — funciona directamente como fichero local

Para publicarlo en **GitHub Pages**:
1. Sube los tres archivos a la rama `main`
2. Ve a **Settings > Pages > Source** y selecciona `main / root`
3. GitHub generará la URL pública automáticamente

## Secciones del dashboard

| Seccion | Contenido |
|---|---|
| Resumen | Stats generales y calendario de publicaciones semanal |
| Identidad y Headline | Datos del perfil, headline optimizado y justificacion |
| Acerca de y CTA | Extracto completo con boton de copia directa |
| Experiencia y SEO | Historial profesional y tabla de keywords |
| Buyer Persona 1 | Marketing Manager — filtros Sales Navigator y pain points |
| Buyer Persona 2 | CEO / Founder — filtros Sales Navigator y pain points |
| Embudo 1 | 3 mensajes para reunion de descubrimiento (Marketing Managers) |
| Embudo 2 | 3 mensajes para descarga + demo (CEOs y Founders) |
| 20 Publicaciones | Posts completos con filtros, expandir y copiar al portapapeles |
| Crecimiento y KPIs | Tacticas de amplificacion y tabla de metricas mensuales |
| Hashtags y Keywords | Sistema de hashtags por volumen con reglas de uso |

---

## Funcionalidades

- **Navegacion lateral** con secciones y sub-secciones
- **Busqueda global** — encuentra cualquier post, mensaje o seccion
- **Filtros de posts** por tipo (Educativo, Opinion, Storytelling, Engagement, Portfolio, Inspiracional, Promocional)
- **Expandir / colapsar** texto de cada publicacion
- **Copiar al portapapeles** — texto del post + hashtags en un solo clic; mensajes de embudo; extracto; CTA
- **Animaciones de navegacion** suaves entre secciones
